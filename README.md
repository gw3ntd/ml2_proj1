# Machine Learning II
## Project 1
### Directory Description
- **bay**: trials of running Bayesian Optimization Tuner
- **bay2**: the updated trials of running Bayesian Optimization Tuner that are included in the final report
- **keras_tn**: Dr. Holsapple's run of the Hyperband Tuner
- **my_logs**: includes the manual tuning runs, specifically in the manual, manual2, and manual3 folders
- **photos**: tensorboard graphics of certain learning curves
- **auto_tuning.ipynb**: the KerasTuner model, with Bayesian Optimization and Hyperband
- **manual_tuning.ipynb**: the random tuner model
- **results.MD**: some results from random tuning
